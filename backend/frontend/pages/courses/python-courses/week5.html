📘 Modules

Modules allow you to reuse code by importing functions from other files. 

import math

print(math.sqrt(16))

🧱 OOP (Object-Oriented Programming)

OOP allows you to structure your code using classes and objects. 

class Person:
    def __init__(self, name):
        self.name = name

p = Person("Nikhil")
print(p.name)
