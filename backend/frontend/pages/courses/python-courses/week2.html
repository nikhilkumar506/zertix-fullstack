📘 Conditions (if-else)

Conditions allow your program to make decisions. Based on certain conditions, different code blocks are executed. 

age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")

🔁 Loops

  • For loop: Repeat fixed times
  • While loop: Repeat until condition is true
for i in range(5):
    print(i)
