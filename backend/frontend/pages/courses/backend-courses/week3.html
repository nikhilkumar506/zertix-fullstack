📘 What is a Database?

A database stores and manages data efficiently. Backend applications use databases to store user information, products, orders, and more.

⚙️ Types of Databases

  • SQL: Structured (MySQL, PostgreSQL)
  • NoSQL: Flexible (MongoDB)

🍃 What is MongoDB?

MongoDB is a NoSQL database that stores data in JSON-like documents. It is flexible and widely used with Node.js.

🔗 CRUD Operations

  • Create: Add data
  • Read: Fetch data
  • Update: Modify data
  • Delete: Remove data

💻 Example Code (Node + MongoDB)

const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost:27017/test');

const userSchema = new mongoose.Schema({
  name: String,
});

const User = mongoose.model('User', userSchema);

User.create({ name: "Nikhil" });
