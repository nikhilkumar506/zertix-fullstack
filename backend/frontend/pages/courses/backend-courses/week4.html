📘 What is Authentication?

Authentication verifies the identity of a user. For example, login using email and password ensures the user is valid.

🪪 What is Authorization?

Authorization determines what actions a user can perform after login.

  • User → Limited access
  • Admin → Full access

🔑 What is JWT?

JWT (JSON Web Token) is used to securely transmit information between client and server.

  • Header
  • Payload
  • Signature

🌐 Authentication Flow

  • User logs in
  • Server validates credentials
  • JWT token generated
  • Token sent to client
  • Client uses token for requests

💻 JWT Example

const jwt = require('jsonwebtoken');

const token = jwt.sign(
  { userId: 1 },
  "secretkey",
  { expiresIn: "1h" }
);

🛡️ Security Best Practices

  • Hash passwords (bcrypt)
  • Use HTTPS
  • Store tokens securely
  • Validate inputs

🌍 Real-World Example

In apps like Instagram or Amazon, JWT is used to keep users logged in securely without storing passwords repeatedly.
