📘 What is Network Security?

Network Security focuses on protecting data during transmission across networks. It ensures that unauthorized users cannot access, modify, or intercept sensitive data.

⚙️ Key Components of Network Security

  • Firewall: Filters incoming and outgoing traffic
  • IDS/IPS: Detects and prevents intrusions
  • VPN: Encrypts communication
  • Proxy: Hides user identity

⚠️ Common Network Attacks

  • Man-in-the-Middle (MITM)
  • Packet Sniffing
  • ARP Spoofing
  • DDoS Attacks

🌍 Real-World Scenario

In a public Wi-Fi network, attackers can intercept your data using packet sniffing tools. If you log into accounts without encryption, your credentials can be stolen easily.

🛠 Wireshark Practical Concept

Wireshark is a network packet analyzer used to capture and inspect data packets. It helps security professionals detect suspicious traffic and analyze attacks.

  • Capture live packets
  • Filter traffic
  • Analyze protocols

🧠 Practical Security Tips

  • Avoid public Wi-Fi for sensitive work
  • Use HTTPS websites only
  • Enable firewall protection
  • Use VPN for secure browsing
