📘 What is Data Analysis?

Data analysis is the process of inspecting, cleaning, transforming, and interpreting data to discover useful insights and support decision-making.

🔄 Data Analysis Lifecycle

  • Data Collection
  • Data Cleaning
  • Data Analysis
  • Visualization
  • Decision Making

🧰 Tools Used

  • Excel: Basic analysis
  • SQL: Database queries
  • Python: Advanced analysis
  • Power BI: Visualization

🌍 Real-World Example

Companies like Amazon analyze customer data to recommend products, improve sales, and understand user behavior.

🧠 Skills Required

  • Problem solving
  • Data understanding
  • Logical thinking
  • Visualization skills

🛠 Practical Mindset

Always ask:

  • What problem am I solving?
  • What does data tell me?
  • How can I improve decisions?

📝 Assignment

Pick any dataset (sales, students, etc.) and:

  • Identify trends
  • Find insights
  • Explain conclusions
