📘 What is SQL?

SQL (Structured Query Language) is used to communicate with databases. It allows analysts to retrieve, filter, and manipulate data efficiently.

⚙️ Core SQL Commands

  • SELECT: Fetch data
  • WHERE: Filter records
  • ORDER BY: Sort data
  • GROUP BY: Aggregate data
  • JOIN: Combine tables

💻 Example Queries

SELECT * FROM users;

SELECT name, age FROM users
WHERE age > 20;

SELECT COUNT(*) FROM orders;

🔗 Understanding JOINs

JOINs are used to combine data from multiple tables:

  • INNER JOIN
  • LEFT JOIN
  • RIGHT JOIN

🌍 Real-World Scenario

In an e-commerce company:

  • Users table stores customer info
  • Orders table stores purchases
  • JOIN is used to connect both
