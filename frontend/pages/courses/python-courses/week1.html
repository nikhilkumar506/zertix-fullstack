📘 What is Python?

Python is a high-level programming language known for its simplicity and readability. It is widely used in web development, data science, AI, and automation.

⚙️ Why Python?

  • Easy to learn
  • Readable syntax
  • Huge community support
  • Used in multiple domains

🧠 Basic Concepts

  • Variables
  • Data Types (int, string, float)
  • Input & Output

💻 Example Code

name = "Nikhil"
age = 20

print("Hello", name)
print("Age:", age)

🌍 Real-World Use

Python is used in companies like Google, Netflix, and Instagram for backend, data processing, and automation.
