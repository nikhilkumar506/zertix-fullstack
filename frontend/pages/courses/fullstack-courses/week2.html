📘 What is Fetch API?

Fetch API is used to send requests from frontend to backend and receive data. It is a core concept in full stack development.

🔄 Request Flow

  • User clicks button
  • Frontend sends request
  • Backend processes data
  • Response sent back
  • Frontend updates UI

💻 Example Code

fetch("https://jsonplaceholder.typicode.com/users")
.then(res => res.json())
.then(data => console.log(data));
