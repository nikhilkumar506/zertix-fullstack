📘 What is Middleware?

Middleware functions are functions that execute during the request-response cycle. They can modify requests, check authentication, log data, or handle errors.

💻 Middleware Example

app.use((req, res, next) => {
  console.log("Request received");
  next();
});

📂 File Upload (Multer)

Multer is a middleware used to handle file uploads in Node.js.

  • Upload images
  • Store files
  • Handle form data
