📘 What is Node.js?

Node.js is a runtime environment that allows JavaScript to run on the server. It is fast, scalable, and widely used for backend development.

⚙️ What is Express.js?

Express.js is a minimal web framework for Node.js that simplifies building APIs and server-side applications.

  • Routing
  • Middleware
  • Request handling

🔗 Creating Your First API

A simple API using Express: 

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send("Hello Backend 🚀");
});

app.listen(3000, () => console.log("Server running"));

🌐 Backend Flow

  • Client sends request
  • Server receives request
  • Logic runs
  • Response sent back
